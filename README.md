# KoaJS & Typescript Starter Project

Requirements:
 * Node v8

Features:
 * Watch mode - server auto-restarts when code changes
 * Koa v2 with #beautiful async functions

Included middleware:
 * koa-router
 * Custom logger with pretty + JSON modes (based on `NODE_ENV`)

## Getting Started

```
npm i
npm run watch-server
```
